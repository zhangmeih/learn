## git的基本操作

### 1. 添加文件

`git add <name>`可以把文件name添加到仓库中

`git add -A ` 是把所有新增，删除都添加到仓库中

### 2.修改文件并提交

* 修改文件
* `git add -A` 添加所有修改
* `git commit -m "描述"`添加对修改的描述
* `git push` 提交代码到github远程仓库
